Authorization
=====================================

Tokens
-------------------------------------

access token
  - a jwt token
  - proves identity when interacting with resources
  - can be used multiple times
  - expires every 15 minutes
refresh token
  - a uuid4 token
  - proves identity when access token is lost or expires
  - one time use
  - expires every 30 days
